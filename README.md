# 시장 사이클 모니터 (v6)

GitHub Pages + Actions 매시간 자동. 4열 반응형.
- 1열 사모대출 신용(최우선): 패널 + EDGAR 8-K + 신용뉴스
- 2열 AI 밸류체인: 반도체·메모리·네트워킹·전력·원자재·하이퍼스케일러 + capex뉴스
- 3열 거시·시스템 위험(위험 측정): VIX·신용스프레드·금리차·광범위시장 + 거시뉴스
- 4열 자금흐름·유동성(돈의 방향): 달러·금리·엔·유가·구리·금·BTC + 자금흐름뉴스 + 기타(규제·추세·IPO)

## 핵심 구분
- 거시(3열) = 위험 판정(정상/주의/위험) → 시스템 종합판정에 반영
- 자금흐름(4열) = 방향만 관찰(판정 안 함)

## 위험 규칙
- 종목: 200일선 아래 OR 당일 −5%↓
- 바스켓: 구성종목 과반 위험이면 ON
- 신용 트리거 뉴스/공시 → 상단 배너 + AI 판정 격상
- 뉴스: 컬럼별 최대 5개, 트리거 우선
