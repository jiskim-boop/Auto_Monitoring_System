# 시장 사이클 모니터

AI 인프라·신용·거시를 한 화면에서 추적하는 자동 갱신 대시보드.
GitHub Actions가 주기적으로 데이터를 수집해 GitHub Pages로 표시.

## 구조
- `index.html` — 대시보드 화면 (data.json을 읽어 렌더)
- `fetch_data.py` — 데이터 수집 (Yahoo 시세 + FRED + EDGAR 8-K + 뉴스 + AI 요약)
- `.github/workflows/update.yml` — 자동 수집 워크플로
- `data.json` — 수집 결과 (자동 생성, git 추적 안 함)

## 화면 구성
- **상단 배너** — 급변·뉴스·공시 + 3대 영역(시스템·AI·신용·버블) 종합 위험 감지
- **3대 판정** — 시장 시스템(타이밍) / AI 사이클(섹터) / 구조적 취약성(배경)
- **한 문장 요약** — AI가 변동성·신용·capex·거시·밸류에이션 종합 브리핑
- **0번 조기경보** — 옵션·신용·가격 3축 가중 합산 (3축 동시=최고경보)
- **1열 신용·유동성** — 사모대출(BDC+레버리지론·CLO 프록시) / 옵션파생 / FRED 유동성
- **2열 AI 밸류체인** — 반도체·메모리·네트워킹·전력·원자재·하이퍼스케일러
- **3열 거시·시스템** — 구조적 버블 / VIX / 신용스프레드 / 금리차 / 한국지수 / 규제 / IPO
- **4열 자금흐름** — 선물 야간세션 / 달러 / 금리 / 엔 / 유가 / 구리 / 금 / BTC

## 설정 (GitHub Secrets 필요)
- `ANTHROPIC_API_KEY` — AI 요약·8-K 본문 요약용
- `FRED_API_KEY` — 유동성·신용 지표용

## 임계값 주의
모든 임계값은 문헌·역사적 기준 기반의 **정성 기준**이며 백테스트로 검증되지 않음.
이 도구는 보조 경보용이며 매매 신호가 아님.
